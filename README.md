# Team Task Manager

## Features
- Authentication
- Project Management
- Task Assignment
- Dashboard
- Role Based Access

## Tech Stack
- React
- FastAPI
- PostgreSQL
- Railway
## Run Backend

cd backend/app
pip install -r requirements.txt
uvicorn main:app --reload

## Run Frontend

cd frontend
npm install
npm run dev
